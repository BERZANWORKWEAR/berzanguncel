Çerez (Cookie) Politikası

Bu Çerez Politikası, BERZAN web sitesini ziyaret ettiğinizde kullanılan çerezlere ilişkin bilgilendirme amacıyla hazırlanmıştır. Çerezler, web sitesinin düzgün çalışması ve kullanıcı deneyiminin geliştirilmesi için kullanılabilir.

1. Çerez Nedir?

Çerezler (cookies), ziyaret ettiğiniz internet siteleri tarafından tarayıcınıza veya cihazınıza kaydedilen küçük metin dosyalarıdır. Çerezler sayesinde site, cihazınızı tanıyabilir ve bazı tercihlerinizi hatırlayabilir.

2. Çerez Türleri

  • Zorunlu Çerezler: Sitenin temel işlevleri için gereklidir.
  • Analitik/Performans Çerezleri: Site kullanımını analiz ederek performansı geliştirmemize yardımcı olur (izin gerektirir).

3. Google Analytics

Web sitemizde, yalnızca açık rızanız bulunması halinde Google Analytics kullanılabilir. “Kabul Et” seçeneğini işaretlemediğiniz sürece analitik çerezler aktif edilmez.

4. Çerezleri Yönetme

Dilediğiniz zaman tarayıcı ayarlarınız üzerinden çerezleri silebilir veya engelleyebilirsiniz. Ancak bazı çerezleri devre dışı bırakmanız halinde web sitesinin bazı bölümleri düzgün çalışmayabilir.
